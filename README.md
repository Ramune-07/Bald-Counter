# Bald Counter (ハゲカウンター)

Bald Counter は、Discordサーバー内で「ハゲ」に関連するワード（ハゲ、薄毛、ピカピカなど）が発言された回数をカウントし、ランキング形式で管理できるエンターテイメント用Discord Botです。

## ✨ 機能

- **自動カウント**: メッセージ内の特定キーワード（正規表現パターン）を検知し、ユーザーごとにカウントを記録します。
- **リアクション**: キーワードを検知したメッセージに対して自動的に「👨‍🦲」のリアクションを付けます。
- **`/count` コマンド**: 自分、または指定したユーザーの「通算ハゲ回数」を確認できます。
- **`/ranking` コマンド**: サーバー内のハゲ発言回数トップ10のランキングを表示します。
- **カスタマイズ可能**: 検知するキーワードは `.env` ファイルから自由に変更・追加が可能です。

## 🛠️ 必要な環境

- Python 3.8 以上
- `discord.py`
- `python-dotenv`

## 🚀 セットアップ方法

1. **リポジトリの準備**
   本プログラムのファイルを任意のディレクトリに配置します。

2. **依存ライブラリのインストール**
   コマンドプロンプトやターミナルで以下のコマンドを実行し、必要なライブラリをインストールします。
   ```bash
   pip install discord.py python-dotenv
   ```

3. **環境変数の設定**
   - 同じディレクトリにある `sample.env` をコピーし、ファイル名を `.env` に変更します。
   - `.env` ファイルをテキストエディタで開き、あなたのDiscord Botのトークンを設定します。
   ```env
   DISCORD_TOKEN=ここにBotのトークンを貼り付け
   ```

4. **Botの起動**
   以下のコマンドでBotを起動します。
   ```bash
   python main.py
   ```
   起動すると SQLite データベース (`hage_counter.db`) が自動的に作成され、スラッシュコマンドが同期されます。

## ⚙️ キーワードのカスタマイズ

デフォルトでは以下のキーワードや絵文字に反応します。
`ハゲ|はげ|禿|薄毛|毛根|むしり|ピカピカ|眩しい|光ってる|ツルツル|👨‍🦲|👴|🥚|🌞`

これを変更したい場合は、`.env` ファイルに `HAGE_WORDS` を設定し、正規表現の OR (`|`) 区切りで好きなワードを指定してください。
```env
# 例: 特定のワードだけにする場合
HAGE_WORDS="ハゲ|はげ|禿"
```

## 📝 スラッシュコマンド一覧

- `/count [user]`: 指定したユーザー（省略時は自分）のハゲカウントを確認します。
- `/ranking`: サーバー内のハゲランキング（トップ10）を表示します。1位には王冠（👑）が付きます。

## 📄 データベースについて
記録はローカルの `hage_counter.db`（SQLite）に保存されます。サーバー（ギルド）ごと、ユーザーごとに独立してカウントされるため、複数のサーバーに導入してもデータが混ざることはありません。
